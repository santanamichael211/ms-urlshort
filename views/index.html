URL Shortener Microservice

By: Michael Santana

Example Usage:

https://ms-urlshort.glitch.me/new/https://www.google.com

https://ms-urlshort.glitch.me/new/https://www.facebook.com


Example Output:

{"original_url":"https://www.google.com","short_url":"https://ms-urlshort.glitch.me/3476"}


Copy link in "short_url" field which will redirect you to the "original_url".
